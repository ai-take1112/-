# Analog-Meter-Reader 
AI（YOLOv8）と画像処理を組み合わせた、アナログ電圧計の自動読み取りシステムです。

## 🌟 プロジェクトの概要
RTSPカメラストリームからリアルタイムで電圧計を検出し、針の角度を解析してデジタル数値として出力します。
大学での研究や、製造現場でのDX（デジタルトランスフォーメーション）を想定したプロトタイプです。

## 🛠 使用技術
- **Python**: YOLOv8による計器検出、OpenCVによる針の角度解析
- **C# (WinForms)**: GUI開発、RTSPストリーム制御、Pythonプロセス連携
- **Library**: Ultralytics, OpenCvSharp, OpenCV-Python

## ✨ こだわったポイント
1. **ハイブリッド構成**: 高度な推論はPython、ユーザー操作はC#と、適材適所で言語を使い分けました。
2. **精度向上**: セグメンテーションマスクを用いて計器の回転補正を行うことで、斜めからの撮影でも正確な読み取りを可能にしました。
3. **リアルタイム性**: 非同期処理（Task/Async）を活用し、解析中も動画が止まらないUIを実現しました。

## 📂 構成
